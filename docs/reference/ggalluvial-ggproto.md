# Base ggproto classes for ggalluvial

Base ggproto classes for ggalluvial

## Format

An object of class `GeomAlluvium` (inherits from `Geom`, `ggproto`,
`gg`) of length 8.

An object of class `GeomFlow` (inherits from `Geom`, `ggproto`, `gg`) of
length 8.

An object of class `GeomLode` (inherits from `Geom`, `ggproto`, `gg`) of
length 6.

An object of class `GeomStratum` (inherits from `GeomRect`, `Geom`,
`ggproto`, `gg`) of length 6.

An object of class `StatAlluvium` (inherits from `Stat`, `ggproto`,
`gg`) of length 6.

An object of class `StatFlow` (inherits from `Stat`, `ggproto`, `gg`) of
length 7.

An object of class `StatStratum` (inherits from `Stat`, `ggproto`, `gg`)
of length 5.
