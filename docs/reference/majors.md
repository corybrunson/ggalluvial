# Students' declared majors across several semesters

This data set follows the major curricula of 10 students across 8
academic semesters. Missing values indicate undeclared majors. The data
were kindly contributed by Dario Bonaretti.

## Format

A data frame with 80 rows and 3 variables:

- `student`:

  student identifier

- `semester`:

  character tag for odd-numbered semesters

- `curriculum`:

  declared major program
