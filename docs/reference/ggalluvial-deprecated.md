# Deprecated functions

These functions are deprecated in the current version and may be removed
in a future version.

## Usage

``` r
is_alluvial(data, ..., silent = FALSE)

is_alluvial_lodes(...)

is_alluvial_alluvia(...)

to_lodes(...)

to_alluvia(...)
```

## Details

Use `is_*_form` instead of `is_alluvial` and `is_alluvial_*`. Use
`to_*_form` instead of `to_*`.
